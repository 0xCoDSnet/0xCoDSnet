**[libfprint — FocalTech FT9201 driver](https://gitlab.freedesktop.org/libfprint/libfprint/-/merge_requests/572)**

Reverse-engineered the FocalTech FT9201 USB fingerprint sensor and wrote a Linux driver from scratch.
Full USB protocol RE from the Windows UMDF driver, custom NCC matching engine.
First open-source driver for this sensor — found in GPD Win 4, AVITA, Gateway, CHUWI and other devices.

`C` `USB Protocol RE` `Ghidra` `Wireshark` `Python`
